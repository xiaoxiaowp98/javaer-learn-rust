# 🦀 Java 开发者的 Rust 学习指南

> 一份面向 Java 开发者的 Rust 完整教程，用你熟悉的 Java 概念作为参照，帮你快速理解 Rust 的核心设计。

## 📖 在线阅读

直接打开 `rust-learn.html` 即可，无需服务器，双击就能看。

## ✨ 特色

- **Java 视角**：每个 Rust 概念都配有 Java 对比，利用你已有的知识快速建立认知
- **代码丰富**：大量可直接运行的代码示例，不是干巴巴的文字说明
- **对比表格**：速查表格式，方便随时翻阅
- **大白话风格**：诙谐幽默，拒绝枯燥
- **单页面设计**：一个 HTML 文件包含全部内容，侧边栏导航，支持键盘翻页（← →）
- **移动端适配**：手机上也能看

## 🛠 技术栈

- 纯 HTML + CSS + JavaScript，无框架依赖
- Prism.js 语法高亮
- 响应式布局，支持桌面和移动端

## 📝 适用人群

- 有 Java 基础的开发者，想学 Rust 但不知道从哪下手
- 正在学 Rust 的 Java 程序员，希望用熟悉的类比加速理解
- 想快速了解 Rust 核心概念的任何开发者
